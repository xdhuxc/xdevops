### xdevops


### 功能

#### 节点管理
1、可以从页面添加node节点到kubernetes系统中。
2、可以批量执行命令
3、可以使用web shell界面登录某台机器进行操作
#### 应用管理
1、以列表或图表形式显示kubernetes常用资源，提供操作界面。
2、显示实时日志。

#### 系统监控
1、从机器层面监控CPU、内存、磁盘，网络IO等，并内置报警策略。
2、监控应用的状态。
